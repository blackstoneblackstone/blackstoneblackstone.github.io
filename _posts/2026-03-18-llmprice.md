## LLM 比价 - 大模型 API 选型利器

接入大模型 API 时，模型多、计费方式不一，比价和选型往往要翻好几家文档。[LLM 大模型价格对比](https://llmprice.cn/)（llmprice.cn）把国内外主流大语言模型的 API 价格集中在一页，用统一单位（元/百万 Token）展示，方便开发者快速对比、控制成本。

## 为什么需要大模型比价？

- **模型多**：OpenAI、Anthropic、Google、Deepseek、阿里云、月之暗面、百川、智谱、零一万物、百度等，每家又有多个模型与版本。
- **计费不统一**：有的按输入/输出分开计价，有的按 Token 或字符，汇率和更新节奏也不一样。
- **选型成本高**：自己逐个查官网费时费力，还容易漏掉新模型或降价信息。

LLM 比价站把「输入价格」「输出价格」做成可排序表格，支持按「国外 / 国内大厂 / 初创公司」筛选，能大大缩短选型时间。

## 核心功能

- **国内外模型覆盖**：国外厂商（如 OpenAI、Anthropic、Google）与国内大厂、初创公司同屏对比。
- **统一计价单位**：按「元/百万 Token」展示，页面注明汇率（如 $1 = ¥7）和更新日期（如 2026-01-29），便于评估真实成本。
- **多维度排序**：支持按输入价格、输出价格升序/降序排列，快速找到便宜或高性价比的模型。
- **分类筛选**：可按「全部 / 国外 / 国内大厂 / 初创公司」缩小范围，适合不同场景的选型需求。

## 适用场景

- **独立开发者**：在预算有限的情况下，为产品挑选性价比合适的模型。
- **技术选型**：为项目或团队做多模型对比，平衡成本与能力。
- **成本监控**：了解各厂商最新定价，便于做用量与预算规划。

## 小结

[llmprice.cn](https://llmprice.cn/) 专注做一件事：把主流大模型 API 的价格放在一起、用统一口径对比。不涉及调用或密钥，只做信息聚合，适合作为选型与比价的常备书签。若你正在选模型或想控制 API 成本，不妨先到这里看一圈再决定用哪家。
